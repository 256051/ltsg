# 🐑 羊了个羊 Web 版

一款基于 Web 的三消休闲游戏，仿「羊了个羊」玩法。

## 🎮 游戏玩法

- 点击上方面块将其移动到下方槽位
- 当槽位中有 **3个相同图案** 的方块时，自动消除
- 消除所有方块即可通关
- 槽位 **7格填满且无法三消** 则游戏失败

## 🛠️ 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 5
- **样式**: CSS3 + Flexbox/Grid

## ✨ 功能特性

- 三层堆叠方块设计，上层方块消除后可点击下层
- 7格槽位，三消自动消除
- 3个道具：洗牌、撤回、移出
- 3个难度递增关卡
- 流畅动画与交互反馈
- 响应式设计，适配 PC 端与移动端

## 🚀 运行项目

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

## 📁 项目结构

```
src/
├── components/      # React 组件
│   ├── Block.jsx        # 单个方块
│   ├── BlockBoard.jsx   # 方块面板
│   ├── SlotBoard.jsx    # 槽位面板
│   ├── ToolBar.jsx       # 道具栏
│   ├── Modal.jsx        # 弹窗
│   └── LevelSelect.jsx  # 关卡选择
├── data/            # 配置数据
│   ├── levels.js        # 关卡配置
│   └── items.js         # 方块图案配置
├── utils/          # 工具函数
│   └── gameLogic.js     # 核心游戏逻辑
├── App.jsx         # 主应用组件
└── App.css         # 全局样式
```
